# githubreadme

*Automates README creation using LLM agents and GitHub analysis.*

## Overview

githubreadme is a Python project that automates the creation of README documentation for GitHub repositories. It uses LLM (Large Language Model) agents to analyze project structure and generate documentation, integrating with services like OpenAI and GitHub for data gathering and natural language generation.

## Tech Stack

- **Primary language:** Python (>=3.12)
- **Main frameworks/libraries:** LangChain, LangGraph, LangGraph Supervisor, python-dotenv, requests, typing
- **External services:** OpenAI (LLM models), GitHub (repository access)

## Architecture

The project is organized across several directories and files:

### Directories

| Path          | Role                                               |
|---------------|----------------------------------------------------|
| agent_tools   | Custom tools for repo structure/analysis           |
| agents        | Agent implementations (doc analyzer, readme writer, etc.) |
| prompts       | Prompt templates for LLM interactions              |
| supervisors   | Supervisory logic/state for orchestration          |

### Notable Files

| Path             | Role                                  |
|------------------|---------------------------------------|
| main.py          | Application entrypoint                |
| .env.example     | Environment variable template         |
| pyproject.toml   | Python project/dependency manifest    |

## Prerequisites

- Python 3.12 or later must be installed.
- An **OpenAI API key** is required to access LLM functionality.
- A **GitHub token** is required, especially when processing private repositories.

Ensure both tokens are set up before running the application.

## Installation

Install the project dependencies by running:

```bash
pip install .
```

(Python 3.12+ required.)

## Configuration

Configuration is managed using environment variables. Add these values to a `.env` file (refer to `.env.example`):

| Variable        | Required | Description                                                  |
|-----------------|----------|-------------------------------------------------------------|
| OPENAI_API_KEY  | Yes      | OpenAI API key (required for language model access)          |
| GITHUB_TOKEN    | Yes      | GitHub token (needed if the repo is private; used in main.py without a fallback/default) |

Use `.env.example` as a template for required values.

## Usage

Run the application with the following command:

```bash
python main.py
```

This will load configuration, initialize LLM agents, run supervisors, and generate README output.
